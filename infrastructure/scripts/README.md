## Назначение

Эта папка предназначена для скриптов, которые помогают работать с репозиторием: подготовка окружения, генерация файлов, служебные операции и автоматизация повторяемых задач

## Типовые сценарии

### Подготовка окружения

Скрипт может устанавливать зависимости, создавать локальные конфигурации и поднимать нужные сервисы

### Генерация файлов

Скрипты могут создавать типовые шаблоны, если это экономит время команды и не засоряет ручные инструкции

### Очистка

Отдельные скрипты могут удалять временные файлы, кэши и локальные артефакты, которые не должны оставаться между запусками

---

## Правила

- скрипты должны быть короткими и предсказуемыми;
- опасные операции требуют явного подтверждения или отдельного имени;
- имена должны объяснять назначение без чтения тела файла;
- каждую новую автоматизацию нужно кратко описывать в README рядом;
- скрипты не должны скрывать критичные действия за неочевидным поведением
